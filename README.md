Resumen
- Colección de notebooks didácticos sobre conceptos básicos de Python orientados al procesamiento de datos.
- Cada cuaderno incluye ejemplos ejecutables, explicaciones y un pequeño examen práctico.
- He añadido un membrete con los datos del autor y una celda con instrucciones/algoritmos y conclusiones en cada cuaderno.

Listado de cuadernos y breve descripción
- 01-Python-Cadenas.ipynb — Operaciones con cadenas (indexación, slicing, concatenación, búsqueda y reemplazo).
- 02-Python-Tuplas.ipynb — Tuplas: creación, indexación, slicing, anidamiento y ordenamiento.
- 03-Python-Listas.ipynb — Listas: modificación, copiado, slicing y métodos útiles (`append`, `pop`, etc.).
- 04-Python-Conjuntos.ipynb — Conjuntos: unicidad, unión, intersección y diferencia.
- 05-Python-Diccionarios.ipynb — Diccionarios: acceso llave→valor, iteración y manipulación.
- 06-Python-Condiciones.ipynb — Condicionales: `if`/`elif`/`else`, operadores comparativos y lógicos.
- 07-Python-Bucles.ipynb — Bucles: `for`, `while`, `range`, `enumerate` y ejemplos prácticos.
- 08-Python-Funciones.ipynb — Funciones: definición, parámetros, retorno y ámbito (local/global).
- 09-Python-Clases.ipynb — Clases y objetos: atributos, métodos, construcción de instancias y ejemplos visuales.
- Practico_Bono_1.ipynb — Ejercicios prácticos para reforzar conceptos mediante funciones y algoritmos.

Cómo abrir y ejecutar
1. Recomendado: usar Jupyter Lab o Jupyter Notebook:

```bash
# desde la carpeta Downloads
jupyter lab
# o
jupyter notebook
```

2. También puedes abrir los notebooks con Visual Studio Code (extensión Jupyter).
3. Ejecuta las celdas en orden (Shift+Enter) y guarda los cambios al finalizar.

Requisitos
- Python 3.8+ (o similar) y Jupyter. No hay dependencias adicionales necesarias para los ejemplos básicos.
- Para algunas celdas gráficas (p.ej. `matplotlib`) asegúrate de tener la librería instalada: `pip install matplotlib`.

Sugerencias de uso
- Ejecuta los ejemplos, modifica variables y vuelve a ejecutar para explorar comportamientos.
- Revisa la celda de "Instrucciones y Algoritmos" que añadí en cada cuaderno para ver los pasos clave y conclusiones.
